| 元素 | 颜色 | 说明 |

### 🎬 动画系统

- ✨ 实时探索动画：看算法如何一步步找路  
- 🏃 姜饼人行走：沿最终路径移动（可爱小帽子与眼睛）  
- ⚡ 可调速度：适合快速演示或慢速学习  
- 🧾 保留探索轨迹：可选择保留搜索过程颜色  

### 📊 详细统计

- 📏 路径长度：从起点到终点的格子数  
- 💰 总代价：考虑地形后的累计体力消耗  
- 👀 访问节点：算法访问过的节点数  
- 🔍 扩展节点：算法处理过的节点数  
- ⏱️ 运行时间：毫秒级计时

## 🚀 快速开始

### 前置要求
- Python 3.8 或更高版本
- 现代浏览器（Chrome / Firefox / Edge / Safari）

### 安装与运行

```bash
# 1. 克隆项目
git clone https://github.com/yedda1117/cookie-Maze.git
cd cookie-Maze

# 2. 创建虚拟环境（推荐）
python -m venv venv

# 3. 激活虚拟环境
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 4. 安装依赖
pip install -r requirements.txt

# 5. 启动应用（入口：main.py）
python main.py
